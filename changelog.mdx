## Deprecations and Migration Guides

Certain legacy endpoints have been deprecated. Update your integrations promptly to avoid disruptions.

<Expandable title="Migrate from v0 to v1 API" default-open="true">
  Replace old endpoint calls with the new structure. Here's an example update:

  <CodeGroup tabs="JavaScript,Python">
  ````javascript
  // Old v0 endpoint
  const response = await fetch('https://api.example.com/v0/workflows');
  
  // New v1 endpoint
  const response = await fetch('https://api.example.com/v1/workflows');
  ````

  ````python
  # Old v0 endpoint
  import requests
  response = requests.get('https://api.example.com/v0/workflows')
  
  # New v1 endpoint
  response = requests.get('https://api.example.com/v1/workflows')
  ````
  </CodeGroup>

  Test your changes in a Codespaces environment before deploying.
</Expandable>
